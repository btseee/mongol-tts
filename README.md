# Монгол Кирилл TTS

![](https://img.shields.io/badge/Бүрэн_дуусаагүй_байна-blue)

Энэхүү төсөл нь **Монгол Кирилл Текстээс Яриа үүсгэх (TTS) систем** бөгөөд **VITS** Архитектурыг ашиглан Монгол хэлээр байгалийн өнгө аястай яриа үүсгэх зорилготой.

## Онцлог
- **VITS архитектур ашигласан**: Төгсгөлийн цэгээс төгсгөлийн Архитектурчлал бүхий хүчирхэг TTS Архитектур.
- **Тэмдэгтийн төвшний сургалт**: Фонем ашиглахгүйгээр шууд Монгол Кирилл үсгүүд дээр сургадаг.
- **Өөрийн датасет ашиглах боломжтой**: Өөрийн Монгол хэлний датасет ашиглан сургалт хийх боломжтой.
- **Олон шатлалт сургалт ба лог бүртгэл**: Архитектурыг тогтмол хадгалж, ахиц дэвшлийг хянах боломжтой.

## Суурилуулалт

Python 3.8+ хувилбарыг болон шаардлагатай хамааралтай сангуудыг суулгаарай.

```sh
pip install -r requirements.txt
```

## Датасет бэлтгэх

1. Өөрийн датасетийг `dataset/` хавтас дотор байрлуулна.
2. Датасет нь дараах бүтэцтэй `metadata.csv` файлтай байна:
   ```csv
   example1.wav|Сайн байна уу?
   example2.wav|Монгол хэлний дуу хоолой.
   ```
3. Аудио файлуудыг `dataset/wavs/` хавтсанд байрлуулна.
- 22050hz , mono

## Архитектур сургах

Сургалтын скриптийг ажиллуулна:

```sh
python train.py
```

## Тохиргоо

`config.py` файлыг засварлах эсвэл `train.py` дахь `VitsConfig`-ийг өөрчилж дараах тохиргоог өөрчлөх боломжтой:
- **Batch size (Багцын хэмжээ)**
- **Epochs (Давталтын тоо)**
- **Шалгалтын өгөгдлийн хуваалт**
- **Тэмдэгтийн багц ба текст цэвэрлэгч**

## Яриа үүсгэх (Inference)

Сургалтаа дууссаны дараа дараах командыг ашиглан яриа үүсгэж болно:

```sh
python TTS/bin/synthesize.py --text "Сайн байна уу?"
```

## Ирээдүйн сайжруулалтууд
- Дуу авианы илэрхийллүүдийг сайжруулах.
- Фонем дээр суурилсан синтез нэмэх.
- Архитектурыг бодит хугацааны (real-time) ашиглахад тохируулах.
